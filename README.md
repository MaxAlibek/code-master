# 🌌 CodeMaster: The Engineering Second Brain

Привет! Я — Алибек, и я верю, что обучение разработке не должно быть хаотичным.

Я создал **CodeMaster**, потому что мне не хватало инструмента, который сочетал бы в себе мощь ИИ и системный подход к знаниям.

Этот проект — часть моей работы в коллективе **Digital Revolution Makers**. Мы создаём технологии, которые помогают инженерам из Узбекистана и всего мира расти быстрее.

---

## 🤔 Почему это важно?

Обычно платформы для обучения либо слишком простые, либо слишком перегруженные.
**CodeMaster** решает это через:

- **Метод PARA** — я интегрировал систему организации жизни (Projects, Areas, Resources, Archives) прямо в учебный процесс.
- **Правило 4/5** — система не даст тебе двигаться дальше, пока ИИ (Gemini) не подтвердит, что ты реально усвоил материал. Это честный путь к мастерству.

---

## 🎨 Эстетика "Cupertino Midnight"

Я фанат минимализма. Я потратил часы, чтобы добиться **Absolute Black (#000000)**.

- **Для глаз** — чисто чёрный цвет идеально смотрится на OLED-экранах и не утомляет в ночные сессии кодинга.
- **Для удобства** — каждая кнопка и каждый отступ выверены по стандартам Apple (44px+), чтобы тебе было удобно учиться даже с телефона, пока ты в дороге.

## 🛠 Что внутри?

Я выбрал инструменты, которые люблю сам:

| Технология | Зачем |
|---|---|
| **FastAPI** | Потому что скорость и асинхронность — это база |
| **React + Tailwind** | Для плавного и отзывчивого интерфейса |
| **Google Gemini AI** | Персональный ментор, который проверяет код в реальном времени |
| **i18n (uz / ru / en)** | Знания не должны иметь границ |

## 🚀 Как запустить у себя

```bash
# 1. Клонируй
git clone https://github.com/MaxAlibek/code-master.git
cd code-master

# 2. Установи зависимости
npm install
pip install -r backend/requirements.txt

# 3. Запускай и становись лучше
npm run dev
```
